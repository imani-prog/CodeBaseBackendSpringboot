API Online

Medilink API

Healthcare management backend service.
All endpoints require JWT authentication except auth routes.

Available Endpoints

POST /api/auth/register Register user
POST /api/auth/login Login & get token
POST /api/auth/refresh Refresh JWT token
GET /api/users/** User management
GET /api/doctors/** Doctor management
GET /api/patients/** Patient management
GET /api/ambulances/** Ambulance tracking
GET /actuator/health Health check

Powered by Spring Boot 3 · Connected to NeonDB
